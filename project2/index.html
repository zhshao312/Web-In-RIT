Differences between Internet & World Wide Web

The Internet is a global system that provides users can share resources and communicate with each other. In some network areas, users can direct access to all the facilities on the Internet such as the colleges. And some network areas, users should get the permission to access to the Internet. Such as the network in companies and network in our home.

The Internet is not synonymous with World Wide Web, the World Wide Web is a way of accessing information over the medium of the Internet. It uses the HTTP protocol, one of the languages spoken over the Internet. Also, the web utilizes browsers such as Chrome and Firefox to gain information.

Origins of the Internet and World Wide Web

The origin of the Internet begins in the 1950s. The original packet networking created in several computer science labs in the United States, Great Britain and France. In the 1960s, the first network named ARPANET using the Internet Protocol was created. The first message was sent by the Computer Science Professer Leonard Kleinrock from his lab to the Stanford Research Institute. Then, in order to doing the packet switching, Tymnet and Telenet were developed in the late 1960s by using plenty of different communications protocols. In the early 1980s, the Internet protocol suite(TCP/IP) was introduced as the standard networking protocol on the ARPANET.

Tim Berners-Lee, on the World Wide Web, found the fact that protocols link hypertext documents into a networking system. By Christmas 1990, Berners-Lee had built all the tools for a working Web, including the HyperText Transfer Protocol(HTTP), the HyperText Markup Language(HTML), the first Web browser(named WorldWideWeb), the first HTTP server software, the first web server, and the first Web pages that described the project itself. The browser could access Usenet and FTP files as well. This is the main history of the Internet and World Wide Web.
