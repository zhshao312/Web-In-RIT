Learning Terminology


Internet:

Internet is the worldwide system that provides different users share and transmit informations by using different internet connecting devices.


World Wide Web:

World Wide Web is a system of Internet servers that support markup language(HTML) documents to link to other documents, also graphics, audio and video files.

More Terminology

Hypertext:

Hypertext is a text which contains links to other texts including graphics, video and sound.

Multimedia:

Multimedia is a medium that using variety of communicative media, such as graphics, drawing, movies, audios and animation.

CLI:

CLI(Command Line Interface) is a user interface in which users response to a prompt by typing in a command line, then receive a respinse back from the system./p>

GUI:

GUI(Graphical user interface) is a type of interface that shows users when they using some software. It has plenty of visual components like buttons, labels, etc.

HCI:

HCI(human-computer interaction) is a research of how people interact with computers. Also, it relates to what extent computers are or are not developed for attracting people successfully.

FTP:

The FTP(File Transfer Protocol) is a normally network protocol which used for transfer data between a client and a server.

SSH:

SSH(Secure Socket Shell) is a network protocol that offers administrators a secure method to connect a remote device.

HTTP:

HTTP(Hypertext Transfer Protocol) is an application protocol used to transfer data over the web. It provides a standard for Web browsers and serves to communicate.

Telnet:

Telnet is a network protocol that provides a user on a computer a way to access to another computer remotely.

HTML:

HTML(Hypertext Markup Language) is a markup language for creating web documents. It is a set of markup tags, each HTML tag can describe different document content.

CSS:

CSS(Cascading Style Sheets) is a style language that describes how HTML elements look like one the screen. It covers fonts, colors, margins, lines, height, weights, background, positions, etc.
